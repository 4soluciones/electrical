jQuery

Los selectores jQuery le permiten seleccionar y manipular elementos HTML.

Todos los selectores en jQuery comienzan con el signo de dólar y paréntesis y se usa para definir / acceder a jQuery$().
La sintaxis básica es: $ ( selector ). accion ()
un "selector" es para "consultar (o buscar)" elementos HTML. luego se define la "accion" jQuery () que se realizará en el elemento (s)
El selector #id
usa el atributo id de una etiqueta HTML para encontrar el elemento específico.
$( document ).ready()
Una página no se puede manipular de forma segura hasta que el documento esté "listo". jQuery detecta este estado de preparación para usted. El código incluido dentro de $ (document) .ready () solo se ejecutará una vez que la página Document Object Model (DOM) esté lista para que se ejecute el código JavaScript.

Eventos del ratón

.click()
pulsar una vez el ratón sobre un elemento
.dblclick()
pulsar el ratón dos veces seguidas sobre un elemento
.mouseenter()
El ratón se sitúa encima de un elemento
.mouseleave()
El ratón, que estaba situado encima de un elemento, sale de él
.hover()
Este evento es una combinación de los dos anteriores
.mousedown()
Momento en que presiona el botón, independientemente de si se suelta o no, vale tanto para el botón izquierdo como para el derecho.
.mouseup()
soltar un botón del ratón después de hacer click. El evento se produce sólo en el momento de soltar el botón
.mousemove()
El evento se produce al mover el ratón sobre un elemento.
.mouseover()
El evento se produce al situarse el ratón sobre un elemento. El evento puede producirse varias veces mientras está encima del elemento.

Eventos del teclado

.keydown()
El evento se produce en el momento que se presiona una tecla, independientemente de si se libera la presión o se mantiene. Se produce una única vez en el momento exacto de la presión; devuelve el número de tecla
.keyup()
El evento se produce en el momento de dejar de presionar una tecla que teníamos pulsada. Los eventos de teclado se suelen aplicar al documento (document), y no a un elemento concreto; devuelve el número de tecla
.keypress()
devuelve el número de caracter.

jQuery AJAX (JavaScript asíncrono y XML)

AJAX
carga datos en segundo plano y mostrarlos en la página web, sin volver a cargar toda la página.
Los métodos jQuery AJAX
puede solicitar texto, HTML, XML o JSON desde un servidor remoto utilizando HTTP Get y HTTP Post - ¡Y puede cargar los datos externos directamente en los elementos HTML seleccionados de su página web!.
Ejemplos de aplicaciones que usan AJAX
pestañas de Gmail, Google Maps, Youtube y Facebook. Sin jQuery, la codificación AJAX puede ser un poco complicada.

Método jQuery load ()

$(selector).load(URL,data,callback);
El parámetro de URL requerido especifica la URL que desea cargar.
El parámetro de datos opcional especifica un conjunto de pares de clave / valor de cadena de consulta para enviar junto con la solicitud.
El parámetro opcional de devolución de llamada es el nombre de una función que se ejecutará después de load()completar el método.
